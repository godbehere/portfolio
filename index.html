Ion Motion Simulator

About Project

This python project was a personal project undertaken to further my knowledge related to mass spectrometry. The simulation program was suggested to me by a scientist who was providing training to me a Sciex. Given a mass charge ratio as well as several other inputs, this application plots the path of an ion in the field of a set of quadrupole electrodes.

What Went Well?

With this project, stability of single ions in a quadrupole can be accurately predicted. I was also able to add in the effect of a magnetic field on ions. I learned a lot while implementing the plotting of the ion path using matplotlib.

What Could Be Improved?

There is no GUI for the user. Inputs have to be hardcoded into the program prior to executing.

Elevator Quoting

About Project

The Elevator Quoting program is an ongoing collaborative project. This project was created in order to test concepts related to efficiently quoting complex mechanical systems. By filling out several simple forms, the user is provided with a cost as well as basic drawing in DXF format. The program is written in C# with the logic and saved quotes stored in a MySQL database.

What Went Well?

The DXF creation was a huge milestone for the team.

What Could Be Improved?

Conversion from imperial to metric units has been disabled currently due to bugs, and needs to be fixed still. More detailed cost breakdown should be included.
